# netflix-clone
<h1>O que é?</h1>
<p>Esse é um projeto simples apenas com uma interface que se assemelha ao netflix e apps semelhantes.</p>

<h2>O que tem no projeto?</h2>
<p>É um projeto extremamente simples em que utilizei a API da TMDB para carregar os posters dos filmes e as outras informações como nome e descrição.</p>

<h2>Tecnologias e resources:</h2>
<li>HTML</li>
<li>CSS</li>
<li>Javascript</li>
<li>Bootstrap 4</li>
<li>Material Icons by Google</li>
<li>https://developers.themoviedb.org/4/getting-started/authorization</li>
